# Movie Recommendation System 🎬

A beginner-friendly AI project that recommends movies based on genres.  
Built with Python, Pandas, and scikit-learn. Can be extended to a web interface with Streamlit.

---

## Features
- Input a movie title and get top 10 similar movies.
- Uses TF-IDF vectorization and cosine similarity on movie genres.
- Lightweight and beginner-friendly.
